    My Param Sheet部品の解説¶

    My Param Sheetの部品¶

    My Param Sheetでは以下の部品を用意しています。

    • レコード

        - _<div style="height:24px"> </div>_               <div style="height:24px"> </div>        <div style="height:24px"> </div>   <div style="height:24px"> </div>   <div style="height:24px"> </div>   <div style="height:24px"> </div>   <div style="height:24px"> </div>   <div style="height:24px"> </div>    <div style="height:24px"> </div>

    • セル

      • ヘッダーセル

        - __ヘッダーセル__

      • タイトルセル

        - _タイトルセル_

      • コードセル
        - `コードセル`
      • リンクセル
        - [リンクセル](#)
      • 取り消しセル
        - ~~取り消しセル~~

      上記にない部品でセルを構成したい場合<div>タグによってセルを表現することが可能です。

      • DIVタグによるセル表現
        - <div  style="height:24px"></div>

    利用方法¶

    My Param Sheetはmy-param-sheetクラスを持つpreの子要素codeの領域内のテキスト定義のみを評価します。
    そのため以下の範囲内にテキスト定義を記載する必要があります。

    • <pre class="my-param-sheet"><code> /* 評価範囲*/ </code><pre>

    以下ではMkDocsを使って執筆する方法を紹介します。

    MkDocsでの利用¶

    MkDcosで記載する場合は以下を記載し、My Param Sheetとして評価するテキスト範囲を指定します。
    テキスト定義には、My Param Sheetの部品が配置できます。 

        ``` my-param-sheet

        /* テキスト定義 */

    ```

    各部品の記載方法(MkDocs)¶

    レコード¶

    行の先頭にハイフン-を指定することでレコードを定義できます。
    記載例は以下の通りです。 

        ``` my-param-sheet
    - _セル_      `セル`      `セル`      `セル`   
    ```

    表示結果

    - _セル_      `セル`      `セル`      `セル`

    ヘッダーセル¶

    レコード行内に__で文字を囲うことでヘッダーセルを定義できます。
    記載例は以下の通りです。 

        ``` my-param-sheet
    - __セル__      __セル__      __セル__      __セル__  
    ```

    表示結果

    - __セル__      __セル__      __セル__      __セル__

    タイトルセル¶

    レコード行内に_で文字を囲うことでタイトルセルを定義できます。
    記載例は以下の通りです。 

        ``` my-param-sheet
    - _セル_     _セル_      _セル_      _セル_  
    ```

    表示結果 

    - _セル_      _セル_     _セル_     _セル_
    コードセル¶

    レコード行内に`で文字を囲うことでコードセルを定義できます。
    記載例は以下の通りです。

        ``` my-param-sheet
    - `セル`     `セル`     `セル`     `セル`  
    ```

    表示結果 

    - `セル`     `セル`     `セル`     `セル`
    リンクセル¶

    レコード行内に[表示文字](リンク情報)を記載することでリンクセルを定義できます。
    記載例は以下の通りです。 

        ``` my-param-sheet
    - [リンク](#)      [リンク](#)      [リンク](#)      [リンク](#)      
    ```

    表示結果 

    - [リンク](#)      [リンク](#)      [リンク](#)      [リンク](#)
    取り消しセル¶

    レコード行内に~~で文字を囲うことで取り消しセルを定義できます。
    記載例は以下の通りです。 

        ``` my-param-sheet  
    - ~~セル~~     ~~セル~~     ~~セル~~     ~~セル~~  
    ```
    表示結果
    - ~~セル~~     ~~セル~~     ~~セル~~     ~~セル~~

    DIVタグによるセル表現¶

    レコード行内にDivタグで文字を囲うことでDivセルを定義できます。
    記載例は以下の通りです。

        ``` my-param-sheet
    - <div>セル</div>      <div>セル</div>      <div>セル</div>      <div>セル</div>      
    ```
    表示結果
    - <div>セル</div>      <div>セル</div>      <div>セル</div>      <div>セル</div>

    グルーピングしたレコードの記載方法(MkDocs)¶

    ハイフン-のインデントを下げることでグルーピングしたレコードを作ることができます。
    記載例は以下の通りです。

    グループレコードに値なし¶

        ``` my-param-sheet
    - __ヘッダ__                           __ヘッダ__      __ヘッダ__      
    - _タイトル_                                                         
        - _タイトル_                       `バリュー`       `バリュー`    
        - _タイトル_                       `バリュー`       `バリュー`    
    ```
    表示結果
    - __ヘッダ__                           __ヘッダ__      __ヘッダ__      
- _タイトル_                                                         
    - _タイトル_                       `バリュー`       `バリュー`    
    - _タイトル_                       `バリュー`       `バリュー`

    グループレコードに値あり¶

        ``` my-param-sheet
    - __ヘッダ__                           __ヘッダ__      __ヘッダ__      
    - _タイトル_                           `バリュー`       `バリュー`    
        - _タイトル_                       `バリュー`       `バリュー`    
        - _タイトル_                       `バリュー`       `バリュー`    
    ```
    表示結果
    - __ヘッダ__                           __ヘッダ__      __ヘッダ__      
- _タイトル_                           `バリュー`       `バリュー`    
    - _タイトル_                       `バリュー`       `バリュー`    
    - _タイトル_                       `バリュー`       `バリュー`

    注意事項¶

    My Param Sheetは1行目を基準に列を構築しています。
    必ず1行目は最大列数を記載してください。
